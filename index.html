Supported Directives

  • .byte: $HEX: Alias for .fill: 1,$HEX;
  • .fill: $N,$HEX: Insert $HEX bytes $N times at current offset;
  • .lbl: $ASCII: Create label at line offset in program's symbol table, use it in completions;
  • .org: $HEX: Override current offset with given origin offset (relative to base address);
